# 部署说明

## 1.起步

### 1.1.注册小程序

首先，你需要前往[微信公众平台][1]注册小程序的账号。

### 1.2.下载微信小程序开发者工具

[官网下载][2]

### 1.3.克隆本项目

你需要克隆本项目到本地，然后通过微信开发者工具导入此项目

### 1.4.开通云开发环境

[官方文档][3]

### 1.5.更改配置文件

配置文件位于`\miniprogram\config.js`

#### 1.5.1.更改云开发ID

修改配置文件内 `ENV` 为你自己的云开发ID
